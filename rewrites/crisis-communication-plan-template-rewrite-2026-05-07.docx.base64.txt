# Crisis Communication Plan Template — Word .docx (base64-encoded)

This file contains the Word document encoded as base64 because GitHub's API for this branch
does not accept binary blobs directly through the channel that produced this commit.

## How to decode and open in Word

**Windows (PowerShell):**
```powershell
$content = Get-Content -Raw '.\crisis-communication-plan-template-rewrite-2026-05-07.docx.base64.txt'
$base64 = ($content -split '## BEGIN BASE64')[1] -split '## END BASE64' | Select -First 1
[IO.File]::WriteAllBytes('.\crisis-communication-plan-template-rewrite-2026-05-07.docx', [Convert]::FromBase64String($base64.Trim()))
Start-Process '.\crisis-communication-plan-template-rewrite-2026-05-07.docx'
```

**macOS/Linux:**
```bash
sed -n '/## BEGIN BASE64/,/## END BASE64/p' crisis-communication-plan-template-rewrite-2026-05-07.docx.base64.txt | sed '1d;$d' | base64 -d > crisis-communication-plan-template-rewrite-2026-05-07.docx
open crisis-communication-plan-template-rewrite-2026-05-07.docx  # macOS
# or: xdg-open ... on Linux
```

**One-line Python (any OS):**
```python
import base64, re
txt = open('crisis-communication-plan-template-rewrite-2026-05-07.docx.base64.txt').read()
b64 = re.search(r'## BEGIN BASE64\s*```(.*?)```', txt, re.DOTALL).group(1).strip()
open('crisis-communication-plan-template-rewrite-2026-05-07.docx','wb').write(base64.b64decode(b64))
```
