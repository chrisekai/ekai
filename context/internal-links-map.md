## Internal Linking Best Practices

When adding internal links to blog content:

1. **Link naturally**. Only link when genuinely relevant and helpful to the reader.
2. **Vary anchor text**. Use different phrasings for the same destination URL across different articles.
3. **3 to 5 links per post**. Aim for 3 to 5 strategic internal links in each blog post; up to 7 for pillar content above 3,000 words.
4. **Deep linking**. Link to specific relevant pages, not just the homepage.
5. **Early links matter**. Links in the first few paragraphs carry more weight.
6. **Update this map**. When new pillar content is published or URLs change, update this file before publishing dependent articles.
